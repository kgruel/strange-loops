Help at --help -v:

deploy — Ship services to production.

Zoom (what to show)
  Controls how much detail is rendered. Stackable: -v for detailed, -vv for full.
  -q, --quiet        Minimal output
  -v, --verbose      Detailed (-v) or full (-vv)

Mode (how to deliver)
  Delivery mechanism. AUTO selects LIVE for TTY, STATIC for pipes.
  -i, --interactive  Interactive TUI
  --static           Static output, no animation
  --live             Live output with in-place updates

Format (serialization)
  Output serialization. ANSI is default for TTY, PLAIN for pipes.
  --json             JSON output
                     Implies --static.
  --plain            Plain text, no ANSI codes
                     Implies --static when piped.

Help
  -h, --help         Show this help
                     Add -v for more detail.

Interaction rules
  --json and --plain imply --static (no animation).
  -q (minimal zoom) implies --static.
  AUTO mode selects LIVE for TTY, STATIC for pipes.
