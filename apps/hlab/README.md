# hlab — homelab monitoring (archived)

> **Note:** This app was an early experiment in building homelab monitoring on top of the strange-loops system. It is not actively maintained and should not be treated as a current example of how to build on the libs. The patterns here predate several architectural changes to the core system.

DSL-driven homelab monitoring and management built on the engine and painted libraries.

See the `libs/` and `apps/loops/` for current usage patterns.
